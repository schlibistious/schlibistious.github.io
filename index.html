WHY THIS WEBSITE EXISTS

------------------------

1. because I'm unemployed

2. because I need money

3. because I want to help people that need help with the most random things needing really random tutorials

4. because I like websites and open source projects.

    But the real question is... WHY DOESN'T THIS WEBSITE EXIST?? We all have those moments in life where we are just in such a random predicament that we just NEED someone to walk them through. That's why I made this project. Me and my helpers can teach you random life skills like how to always open a water bottle first try, the best ways to cut an apple, and so much more.

    I made this project not because I'm just good at doing everything non-chalantly (even though I am), but because I just like doing things like this. Making a frontend website isn't like all other coding languages where it takes 5 hours to debug every new feature you add, instead it just works most of the time. Once you get the hang of each keyword, function, and stop forgetting almost every single CSS property, it's like you have the power to do just evrything on your own. Yes there will be projects where you go insane and end up asking an ai chatbot to help you fix your code, but sometimes that's just the fun of figuring things out. I incourage you to try and make something useful if you code or don't, because when you look back at it and see all you've created, it just feels so good.
